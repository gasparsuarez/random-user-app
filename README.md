## Random User App

 Basic application where random user data is obtained and other differentes functionalities are obtained.

# Instalación

- Clone repository
 
`flutter pub get`\
`flutter run`

# Technologies  

 - Flutter
 - Dart
 - Clean Architecture
 - Cubit + Freezed [State Management] 
 - Dio [Network Client]
 - Get It [Dependency Injection / Service Locator]
